craftRecipe
===========

Defines a crafting recipe.

This block can be soft overridden in scripts.


Hierarchy
---------

**Valid Parent Blocks:**

- :doc:`module`

**Required Child Blocks:**

- :doc:`inputs`

**Possible Child Blocks:**

- :doc:`inputs`
- :doc:`itemmapper`
- :doc:`ouputs`
- :doc:`overlaymapper`


ID Properties
-------------

This block should have an ID.
