whitelist
=========

Whitelists a fluid to be allowed to put inside a `fluid container <https://pz-wiki-modding.github.io/pz-scripts-data/blocks/component/component_fluidcontainer.html>`_.


Hierarchy
---------

**Valid Parent Blocks:**

- :ref:`component-fluidcontainer`


ID Properties
-------------

This block should not have an ID.


Parameters
----------

.. _whitelist-fluid:

fluid
^^^^^

   Type: ``{'main': 'string'}``

A fluid which is allowed to be put inside this fluid container.

   Can be duplicated: ✓
